P02 - Style Guide Template

A template you can use to start building out your own style guides with.

Interactive Elements Text Elements Combined Elements Citations

Interactive Elements

In-text link

Sample code:

<a class="intext_link" href="url"> Text to link</a>

Rendered element:

This is a Link which will take you nowhere.

Navigation Link

CSS selector:

.button-nav with hover animation

Sample code:

<ul id="menu"> <li> <a href="#home" > HOME</a></li> </ul><

Rendered element:

Get Started

Submit/Reset Buttons

Sample code:

<button class="subResBtn" type="submit" value="Submit">Submit</button> <button class="subResBtn" type="reset" value="Reset">Reset</button>

Rendered Element

Image as a Link

Sample code:

<a class="image_link" href="url"> <img alt = "something" src = "image.jpeg" width = "100" height = "100"> </a>
